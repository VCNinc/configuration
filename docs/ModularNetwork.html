ModularNetwork

Class representing a Network. A modular network is principally a set of nodes collectively storing and serving a set of documents (pieces of data). Each node on a modular network covers a particular subset of the network (see mod space). Nodes can join and leave the network or adjust the portion of the network covered by them over time. Nodes covering a particular document are identified using a modulus of the doucment identifier.

minimalCoverageSet(identifiers) → {Object}

Since:

Author:

Returns a minimal set of nodes covering all of the provided identifiers. Attempts to use the fewest nodes possible to cover all of the identifiers specified. Note: The "minimal coverage set" problem in Modular Networks is an instance of the NP-hard "set cover problem" in computer algorithms. The greedy approximation is used here.
Example
let coverage = network.minimalCoverageSet([1234, 2345, 3456, 4567, ...]);
Parameters:
Name Type Description
identifiers Array.<integer> An array of identifiers that should be classified.
Returns:
A set of nodes covering a subset of the provided identifiers (covered) and an array of identifiers which have no coverage (uncovered).

Object
